# Hacker Banner
#Script by Hacky Abinash 

Installation Step :

1) git clone https://github.com/HackyAbinash/Hacker-Banner


2) cd Hacker-Banner


3) chmod +x requirement.sh


4) chmod +x h-ban.sh


5) bash requirement.sh


6) bash h-ban.sh



Uninstallation Step : (Remove Logo From Termux)

1) bash remove.sh

2) Done !!!
